知识点回顾
1. 运算符复习
算术运算符: +, -, *, /, // (整除), % (取余), ** (乘方)
比较运算符: >, <, >=, <=, == (等于), != (不等于)
赋值运算符: =, +=, -=, *=, /=, //=, %=, **=
逻辑运算符: and, or, not
成员运算符: in, not in
身份运算符: is, is not
2. Input 函数
功能: 用于接收键盘输入的内容。
语法: input([提示字符串])提示字符串 (可选): 在接收输入前显示的提示信息。
返回值: 返回用户输入的字符串。 即使输入的是数字，返回的也是字符串类型。
注意事项:input() 函数接收的任何输入都会被解释为字符串。
如果需要将输入用于数学运算，必须使用类型转换函数将其转换为数值类型（如 int(), float()）。
编写代码时，使用的符号必须是英文输入法状态下的符号，如引号，逗号，冒号等。
3. 数据类型转换
int(x): 将 x 转换为整数。
float(x): 将 x 转换为浮点数。
str(x): 将 x 转换为字符串。
4. 字符串操作
字符串连接: 使用 + 运算符连接字符串。
字符串重复: 使用 * 运算符重复字符串。
type(变量): 返回变量的数据类型。
5. True 和 False
True 和 False 是 Python 中的布尔值，用于表示真和假。
注意：True 和 False 的首字母必须大写。小写的 true 和 false 会被解释为变量名。

常见问题解答
什么是input()函数？它有什么作用？
input()函数是Python中用于接收键盘输入内容的函数。它允许程序从用户那里获取数据。当程序执行到input()函数时，它会暂停，等待用户在控制台中输入一些文本，然后用户按下回车键。input()函数会将用户输入的内容作为字符串返回。
input()函数如何使用？
使用input()函数非常简单。你只需要在你的代码中写input()，然后在括号中可以放一个字符串作为提示语，告诉用户应该输入什么。例如：age = input("请您输入年龄: ")。这会在控制台中显示“请您输入年龄: ”，然后等待用户输入。
在使用input()函数时，需要注意哪些细节？
在使用input()函数时，一个重要的细节是确保你使用的标点符号（例如引号、冒号等）是英文输入法状态下的符号。中文输入法下的符号在Python代码中通常会导致错误。此外，input()函数返回的值总是字符串类型，即使你输入的是数字。
input()函数接收到的值是什么数据类型？
input()函数接收到的任何值，无论用户输入的是数字、字母还是其他字符，都会被作为字符串（str）类型处理。这一点非常重要，因为如果你需要对输入进行数值计算，需要先将其转换为整数（int）或浮点数（float类型。
如何将input()函数接收到的字符串转换为数字类型？
要将input()函数返回的字符串转换为整数，可以使用int()函数。例如：age = int(input("请您输入年龄: "))。如果需要转换为浮点数，可以使用float()函数。
为什么需要将input()接收到的字符串进行类型转换？
因为input()函数默认返回字符串类型，如果你想对用户输入的内容进行数学运算，直接使用字符串是无法进行的。你需要先将字符串转换为数值类型（如整数或浮点数），才能进行加减乘除等运算。
如果在字符串连接中使用input()函数返回的值，需要注意什么？
在字符串连接中，如果要把input()函数返回的值和其他字符串连接起来，同样需要注意数据类型。如果input()返回的值没有经过类型转换，可以直接和字符串连接。但是，如果已经转换为数值类型，则需要再次将其转换为字符串类型，例如使用str()函数，才能进行连接。 否则会报错。
True和true有什么区别？ 在Python中，True和False是布尔值的关键字，用于表示真和假。它们的首字母必须大写。如果使用小写的true或false，Python会将其识别为变量名，如果没有定义这些变量，则会导致NameError错误。因此，要注意大小写，确保使用正确的布尔值关键字。
